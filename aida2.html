AIDA 2 初階自由潛水員課程

課程時間

3天

費用

$ 13000 / 人

室內理論學科

  • 認識自由潛水
  • 裝備介紹
  • 耳壓平衡
  • 放鬆與呼吸
  • 自由潛水項目
  • 風險與安全

平靜水域

  • 放鬆及技巧
  • 蛙鞋使用技巧
  • 靜態閉氣 - 2分
  • 動態平潛 - 40米
  • 潛伴救援演練

開放水域

  • 上下岸安全知識
  • 潛伴制度
  • 恆重下潛 - 12米
  • 水下救援 - 5～10米
  • 法蘭茲平壓
  • 攀繩下潛
  • 躬身入水
  • 安全轉身姿勢

費用包含

  • 基本装備使用
    ( 面鏡、呼吸管、防寒衣、長蛙鞋、配重帶 )
  • 泳池使用費
  • 保險
  • 國際證照費 ( 通過即可發證 )

報名資格

  • 需年滿18歲以上 ( 16-17歲需監護人同意證明 )
  • 身心健康、無心血管、呼吸道疾病。
    若有重大疾病者、需由醫生簽署同意證明。
